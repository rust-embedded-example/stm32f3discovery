创建PAC crate

pac 是使用svd2rust工具处理芯片的svd文件得到的。

获取stm32f303的svd文件

推荐从官网获取,点击此处,开始下载。将跳转到ST官网并进行下载。下载完成后得到一个压缩包，解压并找到STM32F303.svd文件。

创建stm32f303pac crate

在根目录下创建一个crate：

cargo new stm32f303pac

将STM32F303.svd文件移动到`./stm32f303pac/下。

安装svd2rust

svd2rust用以提取STM32F303.svd文件中的芯片信息，并生成crate文件。运行：

cargo install svd2rust

安装cargo-form

安装cargo-form用以格式化pac crate，运行：

cargo install form

生成pac crate

在路径./stm32f303pac/下运行：

svd2rust -i STM32F303.svd
rm -rf src
form -i lib.rs -o src/ && rm lib.rs
cargo fmt

修改./stm32f303pac/Cargo.toml

需要修改./stm32f303pac/Cargo.toml，使得启用rt,更方便使用这个crate。修改后如下：

[package]
name = "stm32f303pac"
version = "0.1.0"
edition = "2024"

[dependencies]
critical-section = { version = "1.2", optional = true }
cortex-m = "0.7.7"
cortex-m-rt = { version = "0.7.5", optional = true }
vcell = "0.1.3"

[features]
rt = ["cortex-m-rt/device"]

编译

在./stm32f303pac/路径下运行:

cargo build

若有错误与告警，按提示修复错误与告警，可以使用AI辅助。

完成后得到一个stm32f303pac crate。
